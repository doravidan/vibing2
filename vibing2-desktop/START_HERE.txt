┌────────────────────────────────────────────────────────────────┐
│  ESSENTIAL COMMANDS                                            │
└────────────────────────────────────────────────────────────────┘

# Build DMG locally (no signing)
pnpm run build:dmg

# Sign and notarize (requires Apple Developer account)
pnpm run sign

# Full release (build + sign)
pnpm run release

# Version management
pnpm run version:patch    # 1.0.0 → 1.0.1
pnpm run version:minor    # 1.0.0 → 1.1.0
pnpm run version:major    # 1.0.0 → 2.0.0

┌────────────────────────────────────────────────────────────────┐
│  FIRST-TIME SETUP                                              │
└────────────────────────────────────────────────────────────────┘

Local builds (no signing):
  → No setup needed! Just run: pnpm run build:dmg

Code signing (optional, for distribution):
  1. Join Apple Developer Program ($99/year)
  2. Create Developer ID Application certificate
  3. Generate app-specific password
  4. Copy .env.production.example to .env.production
  5. Add your credentials

GitHub Actions (optional, for automation):
  → Follow GITHUB_SECRETS_SETUP.md

Full details in: BUILD_QUICK_START.md
