DESIGN PRINCIPLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SIMPLICITY
  ✓ Clean, uncluttered design
  ✓ Recognizable at small sizes
  ✓ Focused on core brand mark (V2)

SCALABILITY
  ✓ Works from 16x16 to 1024x1024
  ✓ Elements adjust based on size
  ✓ Core mark always visible

BRAND CONSISTENCY
  ✓ Matches web app colors (purple/pink)
  ✓ V2 represents Vibing2
  ✓ Professional yet approachable

ACCESSIBILITY
  ✓ High contrast (white on gradient)
  ✓ Works on light/dark backgrounds
  ✓ Color-blind friendly
  ✓ Recognizable in grayscale

PLATFORM NATIVE
  ✓ Follows macOS design guidelines
  ✓ Rounded corners (22% radius)
  ✓ Appropriate visual weight
  ✓ Professional appearance


TECHNICAL SPECIFICATIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Format:     PNG (24-bit RGB + 8-bit Alpha)
Color:      sRGB color space
DPI:        72 (standard for screen)
Sizes:      16, 32, 64, 128, 256, 512, 1024 (all @1x and @2x)
Bundle:     .icns (macOS), .ico (Windows)
Canvas:     Square (1:1 aspect ratio)
Corners:    Rounded (225px @ 1024x1024)
Safe Area:  90% of canvas (922x922 @ 1024x1024)


NEXT STEPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GENERATE ICON
   □ Open generate-icon.html in browser
   □ Click "Download All Icons"
   □ Save 1024x1024 as icon.png

2. CREATE PLATFORM ICONS
   □ Run: pnpm tauri icon src-tauri/icons/icon.png
   □ Verify all sizes generated
   □ Check icon.icns created

3. TEST ICON
   □ Build app: pnpm tauri build
   □ Install .app bundle
   □ Check Finder appearance
   □ Check Dock appearance
   □ Test light/dark mode

4. OPTIMIZE (Optional)
   □ Use ImageOptim or pngquant
   □ Verify file sizes
   □ Re-test appearance
   □ Compare with original

5. PREPARE FOR RELEASE
   □ Document design decisions
   □ Create style guide
   □ Archive source files
   □ Plan for updates
