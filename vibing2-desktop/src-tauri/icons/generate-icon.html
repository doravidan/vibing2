How to Use These Icons

Step 1: Generate Icons

Click "Download All Icons" above to get all required sizes.

Step 2: Use Tauri Icon Command

Save the 1024x1024 master icon as icon.png, then run:

cd vibing2-desktop
pnpm tauri icon src-tauri/icons/icon.png

This will automatically generate all platform-specific icons:

  • 32x32.png - Small icon
  • 128x128.png - Standard icon
  • 128x128@2x.png - Retina icon (256x256)
  • icon.icns - macOS bundle icon
  • icon.ico - Windows icon

Step 3: macOS App Store Requirements

For App Store submission, you need:

  • Icon sizes: 16x16, 32x32, 64x64, 128x128, 256x256, 512x512, 1024x1024 (all @1x and @2x)
  • Format: PNG with transparency or ICNS bundle
  • Design: No transparency in corners (use rounded rect background)
  • Safe area: Keep important elements within 90% of canvas
  • Consistency: Icon should look good on light and dark backgrounds

macOS Bundle Configuration

The icon is already configured in tauri.conf.json:

"bundle": {
  "icon": ["icons/icon.png"],
  "category": "DeveloperTool"
}

Professional Icon Design Tips

  • Simplicity: Icons should be simple and recognizable at 16x16
  • Contrast: Ensure good contrast for accessibility
  • Consistency: Match your brand colors and style
  • Testing: Test icon at all sizes and on different backgrounds
  • Platform guidelines: Follow Apple Human Interface Guidelines for macOS
  • File size: Optimize PNGs (use tools like pngquant or ImageOptim)

Alternative: Professional Design Tools

For production apps, consider using:

  • Figma: Design icon with components, export at all sizes
  • Sketch: Native macOS app with icon templates
  • Adobe Illustrator: Vector design with artboards for each size
  • SF Symbols: For macOS-native look (requires macOS 11+)
